## Frontend – Run Instructions (Windows / PowerShell)

Run all commands from the project root: `h:\work\ptj\playground\flashy-dashboard`.

```powershell
# 1) Go to project root
cd h:\work\ptj\playground\flashy-dashboard

# 2) Go to frontend folder
cd frontend

# 3) Install dependencies (first time or when package.json changes)
npm install

# 4) Run Vite dev server
npm run dev
```
